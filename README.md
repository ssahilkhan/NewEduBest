# NewEduBest

An advanced educational technology platform built with React and Vite, featuring a visual editor plugin for interactive content creation. Designed to deliver a modern, responsive learning experience.

## Progress

![Progress](https://progress-bar.dev/50)

## Tech Stack

- **React 19** (Vite) - Frontend framework
- **Radix UI** - Accessible component primitives (Alert Dialog, Avatar, Checkbox, Dialog, Dropdown Menu, Label, Select, Slider, Tabs, Toast)
- **Tailwind CSS** + 	ailwindcss-animate - Utility-first styling
- **Framer Motion** - Animations
- **Lucide React** - Icon library
- **React Router DOM** - Client-side routing
- **class-variance-authority** + **clsx** + **tailwind-merge** - Styling utilities
- **Visual Editor Plugin** - Interactive content authoring

## Future Scope

- Rich text editor with drag-and-drop content blocks
- Student progress tracking and analytics
- Quiz and assessment engine
- Multi-language internationalization
- Collaborative editing in real-time
- Mobile-responsive PWA
